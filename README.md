## Clone prj
Khi kéo code về nhớ cài đặt nodejs để sử dụng npm:
[Nodejs](https://nodejs.org/en/)

sau đó vào cmd dùng lệnh sau để cài đặt hết các deps cần thiết:
```
npm install
```

cuối cùng thì chạy project
```
npm start
```
